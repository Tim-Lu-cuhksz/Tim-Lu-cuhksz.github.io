Leaderboard Simulation via LLM Agents

To access the simulation website, please contact d72@uwaterloo.ca beforehand to set up the server (for now the website is deployed locally only).

Click link to access the simulator once the server is set up.

Settings

We use gpt-4o-mini to simulate different roles including CTO, CFO, and COO.

We set the number of rounds to 6 where different roles take turns to express their opinions. Note that the interface displays the responses only when the backend finishes all 6 rounds.
